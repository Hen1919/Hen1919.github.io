Materiales Utilizados

Cartón: Base de la maqueta y soporte estructural.

Canicas: Usadas para representar la tercera ley de Newton.

Coche: Representa el objeto en movimiento en el plano inclinado.

Polea: Permite el movimiento del sistema con cuerda.

Balón: Representa el objeto en reposo para la primera ley.

Proceso

1

Diseño de la maqueta

Se planifica la maqueta incluyendo las tres leyes de Newton y sus elementos representativos.

2

Recolección de materiales

Se recogen cajas, cartón, figuras, cuerdas, poleas y demás elementos para armar la maqueta.

3

Construcción

Se arman las escenas para cada ley: el jugador con el balón, el plano inclinado con polea y las canicas.

4

Presentación

Se prepara la exposición, se realizan cálculos y se crea la página web con toda la información.
